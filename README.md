# Модель предсказания колец Лизеганга 

# Автор модели - Прошин Никита Владимирович, Россия, Москва, 2026

Данный репозиторий содержит код и данные для двухстадийной модели машинного обучения, предсказывающей образование периодических осадков (колец Лизеганга) в гелях и их геометрический параметр — spacing coefficient p.

## Структура репозитория

- `Proshin_liesegang_rings_model.ipynb` — основной код (Jupyter Notebook для Google Colab)
- `liesegang_dataset.xlsx` — собранный датасет (213 экспериментов, 210 после очистки)
- `Документ Microsoft Word.docx` — диссертация с описанием методологии и результатов
- `README.md` — этот файл

## Запуск модели в Google Colab

1. Откройте файл `Proshin_liesegang_rings_model.ipynb` в Google Colab.
2. Запустите первую ячейку — появится кнопка для загрузки датасета.
3. Загрузите файл `liesegang_dataset.xlsx`.
4. Дождитесь расчёта признаков и обучения модели (около 10–20 секунд).
5. После появления интерактивной формы выберите режим:
   - **Использовать обученную модель** — предсказание на основе обученных коэффициентов.
   - **Ручной расчёт (свои коэффициенты)** — позволяет ввести любые коэффициенты классификатора и регрессии.
6. Введите все запрашиваемые параметры (см. ниже) и нажмите «ПРЕДСКАЗАТЬ».

## Входные параметры (что нужно вводить пользователю)

### Справочные данные для ионов (находятся в таблицах физико-химических свойств)

| Параметр | Обозначение | Пример для Ag⁺ | Пример для Cr₂O₇²⁻ | Где взять |
|----------|-------------|----------------|--------------------|------------|
| D_water иона ВНУТРИ (м²/с) | D_in | 1.65e-9 | — | CRC Handbook |
| D_water иона СНАРУЖИ (м²/с) | D_out | — | 1.00e-9 | CRC Handbook |
| Радиус иона ВНУТРИ (Å) | r_in | 1.15 | — | Shannon (1976) |
| Радиус иона СНАРУЖИ (Å) | r_out | — | 2.50 | Shannon (1976) |
| Заряд иона ВНУТРИ | z_in | 1 | — | Периодическая таблица |
| Заряд иона СНАРУЖИ | z_out | — | 2 | Периодическая таблица |
| Стехиометрический коэффициент иона ВНУТРИ | nu_in | 2 (для Ag₂Cr₂O₇) | — | Формула осадка |
| Стехиометрический коэффициент иона СНАРУЖИ | nu_out | — | 1 (для Ag₂Cr₂O₇) | Формула осадка |

**Важно:** Стехиометрические коэффициенты nu_in и nu_out — это числа ионов в формуле осадка. Например, для Ag₂Cr₂O₇: nu_in (Ag⁺) = 2, nu_out (Cr₂O₇²⁻) = 1.

### Экспериментальные параметры

| Параметр | Обозначение | Пример | Примечание |
|----------|-------------|--------|-------------|
| C_in (М) | C_in | 0.1 | Концентрация внутреннего электролита (в геле) |
| C_out (М) | C_out | 0.1 | Концентрация внешнего электролита (раствор сверху) |
| Тип геля | gel_type | желатин | желатин / агароза / силикагель |
| Концентрация геля (%) | gel_conc | 5 | Для желатина обычно 3–10%, для агарозы 1–2% |
| Температура (°C) | T | 22 | Большинство экспериментов при комнатной |
| pH | pH | 7.0 | Важно для гидроксидов и фосфатов |
| E (В/м) | E | 0 | Постоянное электрическое поле (0, если нет) |

## Интерпретация результатов

- **Вероятность образования колец P** — значение от 0 до 1. Если P ≥ 0.5, модель предсказывает наличие колец. **Внимание:** из-за дисбаланса классов (197 положительных, 13 отрицательных) AUC = 0.636, поэтому вероятность не следует интерпретировать как точную меру воспроизводимости. Полагайтесь только на бинарный ответ (кольца есть/нет).
- **Spacing coefficient p** — предсказанное отношение расстояний между соседними кольцами. MAE = 0.0213, MAPE = 2.03%. Чем ближе p к 1, тем более равномерно расположены кольца.

## Пример предсказания

Для системы Ag₂Cr₂O₇ (желатин 5%, C_in=0.1 M, C_out=0.1 M, T=22°C, pH=7, E=0):
- Вводим: D_in=1.65e-9, D_out=1.00e-9, r_in=1.15, r_out=2.50, z_in=1, z_out=2, nu_in=2, nu_out=1.
- Модель выдаёт: P ≈ 0.96 → кольца образуются, p ≈ 1.075.

## Ограничения модели

- Классификатор обучен на сильно несбалансированном датасете (197 положительных, 13 отрицательных). AUC = 0.636.
- Регрессия объясняет лишь 42.5% дисперсии p (R² = 0.425). Неучтённые факторы: размер пор геля, время старения, комплексообразование.
- Модель применима только для реакций ионного осаждения в гелях. Не использовать для редокс-систем, систем с сильным комплексообразованием и безгелевых сред.
